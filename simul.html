Simulación de Sistemas

El análisis de elementos finitos (FEA) es un método computarizado para predecir cómo reaccionará un producto ante las fuerzas, la vibración, el calor, el flujo de fluidos y otros efectos físicos del mundo real.

El análisis de elementos finitos muestra si un producto se romperá, desgastará o funcionará como se espera. Se denomina análisis, pero en el proceso de desarrollo de productos, se utiliza para predecir qué ocurrirá cuando se utilice un producto.

FEA descompone un objeto real en un gran número (entre miles y cientos de miles) de elementos finitos, como pequeños cubos. Las ecuaciones matemáticas permiten predecir el comportamiento de cada elemento. Luego, una computadora suma todos los comportamientos individuales para predecir el comportamiento real del objeto.

El análisis de elementos finitos predice el comportamiento de los productos afectados por una variedad de efectos físicos, entre los que se incluyen:

● Esfuerzo mecánico
● Vibración mecánica
● Fatiga
● Movimiento
● Transferencia de calor
● Flujo de fluidos
● Electrostática
● Moldeado por inyección de plástico
